# Walkthrough - Personal Portfolio Website

## Overview
A modern, responsive personal portfolio website built with React and Vite. The site features a dark theme, smooth page transitions, and interactive elements.

## Features
- **Single Page Layout**: All content is now on a single page with smooth scrolling navigation.
- **Home Section**: Hero section with a brief overview and profile image.
- **Projects Section**: Expandable project cards with detailed information, real project images, and improved action buttons.
- **About Me Section**: Detailed sections for Journey, Experiences, Skills, and Contact.
- **Global**: Responsive navigation, smooth animations using Framer Motion, and a professional design.

## How to Run
1. Navigate to the project directory:
   ```bash
   cd shravani-portfolio
   ```
2. Install dependencies (if not already done):
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Open your browser at the URL shown (usually `http://localhost:5174`).
